How to play

Wordle is a word-guessing game where players try to guess a hidden word within six attempts.
The game provides feedback for each guess to help players find the correct word.

Wordle Rules:

Each guess must be a valid five-letter word. After submitting a guess, the game provides
color-coded feedback:

  • 🟩 Green: The letter is in the correct position.
  • 🟨 Yellow: The letter is in the word but in the wrong position.
  • ⬜ Gray: The letter is not in the word at all.
Players must use the feedback to refine their guesses and discover the hidden word.
The goal is to guess the word within six attempts.
Have fun and challenge yourself to find the word as quickly as possible!
