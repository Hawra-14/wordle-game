# Kawaii Wordle Game

_A super cute wordle game_

![Screenshot of my game](images/Wordle-game.png)


## Getting Started

### Play the Game 
[Deployed Game](https://hawra-14.github.io/wordle-game/)

### How to Play
1. Think of a five-letter word and type it using the on-screen keyboard
2. Press ENTER to submit your guess
3. The game will provide feedback on your guess:
   - 🟩 Green: The letter is in the correct position.
   - 🟨 Yellow: The letter is in the word but in the wrong position.
   - ⬜ Gray: The letter is not in the word at all.
4. Use the feedback to refine your guesses and discover the hidden word
5. The goal is to guess the word within six attempts. Good luck! 🍀

### Installation
No installation required! Simply clone the repo to your machine and open the `index.html` file in your browser.

```
git clone https://github.com/Hawra-14/wordle-game
cd wordle-game
open index.html
```
